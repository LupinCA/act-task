# ActTask

Aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Marcar como concluídas
- Remover tarefas
- Persistência de dados com localStorage
- Adicionar tarefas com Enter

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Projeto criado para praticar manipulação de DOM, eventos, lógica com arrays e persistência de dados no navegador.
